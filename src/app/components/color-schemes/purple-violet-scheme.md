# Purple + Violet Color Scheme (Kreativní & Profesionální)

## Primary Colors
- **Purple**: purple-50, purple-100, purple-200, purple-400, purple-500, purple-600, purple-700, purple-800, purple-900, purple-950
- **Violet**: violet-50, violet-100, violet-200, violet-400, violet-500, violet-600, violet-700, violet-800, violet-900, violet-950

## Accent Colors
- **Indigo**: indigo-100, indigo-400, indigo-500, indigo-600, indigo-700, indigo-900 (Backend)
- **Fuchsia**: fuchsia-100, fuchsia-400, fuchsia-600, fuchsia-900 (Tools & Design)
- **Rose**: rose-100, rose-400, rose-600, rose-900 (Email)

## Gradients
- Hero: `from-violet-50 to-purple-50` (light) / `from-gray-900 to-purple-950` (dark)
- Avatar: `from-purple-600 to-violet-600`
- Contact: `from-violet-50 to-purple-100` (light) / `from-gray-900 to-purple-950` (dark)
- Philosophy: `from-violet-50 to-purple-50` (light) / `from-violet-950 to-purple-950` (dark)
- Projects Images: `from-purple-100 to-violet-100` (light) / `from-purple-900 to-violet-900` (dark)
- Availability Card: `from-purple-600 to-violet-600`

## Borders & Highlights
- Featured cards: `border-purple-200` / `dark:border-purple-800`
- Current badge: `bg-green-600`
- Project borders: `border-purple-200` / `dark:border-purple-800`

## Pocit & Asociace
- **Pocit**: Kreativní, inovativní, prémiový, umělecký
- **Asociace**: Design thinking, UI/UX expertise, kreativita, inovace
- **Vhodné pro**: FE architect s důrazem na design a UX zkušenosti

## Usage Map
- **Hero section**: Violet-purple gradient background, purple-violet avatar gradient
- **Skills section**: Purple pro Frontend, Indigo pro Backend, Fuchsia pro Tools
- **Experience section**: Purple accents, green "Current" badge
- **Projects section**: Purple-violet gradient na obrázky
- **About section**: Purple accents, violet-purple philosophy section
- **Contact section**: Violet-purple gradients, purple-violet CTA card
- **Social icons**: Rose pro Email, Violet pro LinkedIn, Purple pro GitHub

## Design Philosophy
Tato paleta zdůrazňuje kreativitu a inovaci, ideální pro prezentaci software architekta 
s silným zaměřením na front-end a UI/UX. Fialové tóny asociují:
- Kreativní myšlení
- Premium kvalitu
- Moderní design approach
- Inovativní řešení
