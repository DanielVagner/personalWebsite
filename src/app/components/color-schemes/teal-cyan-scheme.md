# Teal + Cyan Color Scheme (Moderní Tech)

## Primary Colors
- **Teal**: teal-50, teal-100, teal-200, teal-400, teal-500, teal-600, teal-700, teal-800, teal-900, teal-950
- **Cyan**: cyan-50, cyan-100, cyan-200, cyan-400, cyan-500, cyan-600, cyan-700, cyan-800, cyan-900, cyan-950

## Accent Colors
- **Blue**: blue-100, blue-400, blue-500, blue-600, blue-700, blue-900 (Backend)
- **Cyan**: cyan-100, cyan-400, cyan-600, cyan-900 (Tools & Design)
- **Amber/Yellow**: amber-50, amber-100, amber-400, amber-500, amber-600, amber-900, yellow-500 (Warm accent - Highlights, CTA)
- **Orange**: orange-100, orange-400, orange-600, orange-900 (Email)

## Gradients
- Hero: `from-cyan-50 to-teal-50` (light) / `from-gray-900 to-teal-950` (dark)
- Avatar: `from-teal-600 to-cyan-600`
- Contact: `from-cyan-50 to-teal-100` (light) / `from-gray-900 to-teal-950` (dark)
- Philosophy: `from-cyan-50 to-teal-50` (light) / `from-cyan-950 to-teal-950` (dark)
- Projects Images: `from-teal-100 to-cyan-100` (light) / `from-teal-900 to-cyan-900` (dark)
- Availability Card: `from-amber-500 to-yellow-500` ✨ (Warm gold accent)

## Borders & Highlights
- Featured cards: `border-teal-200` / `dark:border-teal-800`
- Current badge: `bg-green-600`
- Project borders: `border-teal-200` / `dark:border-teal-800`

## Pocit & Asociace
- **Pocit**: Moderní, technologický, svěží, dynamický
- **Asociace**: Cloud computing, moderní dev tools, SaaS, startups
- **Vhodné pro**: FE architect s moderním přístupem, tech-forward profesionály
- **Zlatý akcent**: Prémiový, profesionální, důraz na kvalitu

## Usage Map
- **Hero section**: Cyan-teal gradient background, teal-cyan avatar gradient
- **Skills section**: Teal pro Frontend, Blue pro Backend, Cyan pro Tools
- **Experience section**: Teal accents, green "Current" badge
- **Projects section**: Teal-cyan gradient na obrázky
- **About section**: Amber/Gold accents na highlights, cyan-teal philosophy section
- **Contact section**: Cyan-teal gradients, **amber-yellow CTA card** (zlatý akcent)
- **Social icons**: Orange pro Email, Cyan pro LinkedIn, Teal pro GitHub

## Design Philosophy
Tato paleta evokuje moderní technologie a inovaci s teplým zlatým akcentem:
- Svěží, moderní vzhled (teal + cyan)
- Asociace s cloud platformami (AWS, Azure)
- Tech-savvy profesionalita
- Dynamický a progresivní approach
- **Zlatý akcent** přidává teplo a prémiový pocit
- Vhodné pro startups a moderní tech společnosti
