# Original Blue-Indigo Color Scheme

## Primary Colors
- **Blue**: blue-50, blue-100, blue-200, blue-500, blue-600, blue-700
- **Indigo**: indigo-50, indigo-100, indigo-600, indigo-950

## Accent Colors
- **Purple**: purple-100, purple-500, purple-600, purple-700, purple-900 (Backend)
- **Green**: green-100, green-400, green-600, green-900 (Tools)
- **Red**: red-100, red-400, red-600, red-900 (Email)

## Gradients
- Hero: `from-blue-50 to-indigo-100` (light) / `from-gray-900 to-gray-800` (dark)
- Avatar: `from-blue-500 to-indigo-600`
- Contact: `from-blue-50 to-indigo-100` (light) / `from-gray-900 to-gray-800` (dark)
- Philosophy: `from-blue-50 to-indigo-50` (light) / `from-blue-950 to-indigo-950` (dark)
- Availability Card: `from-blue-500 to-indigo-600`

## Borders & Highlights
- Featured cards: `border-blue-200` / `dark:border-blue-800`
- Current badge: `bg-green-600`

## Usage Map
- **Hero section**: Blue gradient background, blue-indigo avatar gradient
- **Skills section**: Blue for Frontend, Purple for Backend, Green for Tools
- **Experience section**: Blue accents, green "Current" badge
- **Projects section**: Blue-purple gradient on images
- **About section**: Blue accents, blue-indigo philosophy section
- **Contact section**: Blue-indigo gradients, blue-indigo CTA card
